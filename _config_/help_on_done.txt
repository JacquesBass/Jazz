
  Congratulations, you have succesfully configured your project. Now,
you should be able to:

  1. Build the server

  - cd server
  - make                // Shows help when invoked without arguments
  - make jazz           // Compiles the server in RELEASE mode
  - sudo make install   // Installs the compiled server as a Linux service

  2. Build the R package

  - cd r_package
  - ./build.sh

  3. Build the Python package

  - cd py_package
  - ./build.sh
